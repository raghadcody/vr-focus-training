## Project Overview

**Problem:**  
Children with ADHD often face challenges with executive functions such as planning, sequencing, inhibition control, sustained attention, organization, and self-regulation. These difficulties impact daily living activities, including morning routines like waking up, personal hygiene, eating, cleaning, and staying focused.

**Solution:**  
**My Super Morning VR** is an immersive Virtual Reality system designed for clinics, therapists, schools, and parents. It guides children step-by-step through realistic morning routines:

- Waking up & making the bed  
- Brushing teeth  
- Dressing & packing their backpack  
- Eating breakfast  
- Washing dishes & sorting trash  
- Managing unexpected challenges (e.g., cleaning after a pet)  

The VR experience provides structured practice, adaptive prompts, multimodal feedback, difficulty regulation, and a reward system using praise and stars. Each activity intentionally trains different executive functions.

**Unique Value Proposition:**

- Scientifically validated ADHD-centered VR experience  
- Teaches independent real-life skills through guided repetition  
- Includes ‚ÄúCatch the Comet,‚Äù a measurable focus-training mini-game  
- Aligns with UDL, ISTE, WCAG/508, pediatric OT, and ADHD clinical guidelines  
- Clinic-ready: logs performance, errors, progress, attention levels, and distractions

**Primary End Users:**

1. **Clinics & Therapists**  
   - Need measurable progress indicators  
   - Require Executive Function development tools  
   - Want customizable therapy plans  

2. **Schools**  
   - Inclusive, UDL-aligned learning tools  
   - Structured routines and behavioral practice  

3. **Parents**  
   - Support for morning routine independence  
   - Safe, engaging training outside therapy sessions  

4. **Children (ages 6‚Äì12 with ADHD)**  
   - Short loops, fun reinforcement, predictable sequences  
   - Gentle guidance and low-stress practice  

**Key Features:**

- **Full Morning Routine Simulation:** Builds real-world independence and sequencing habits  
- **Adaptive Difficulty + Guided/Independent Modes:** Scales cognitive load, adjusts assistance  
- **Mom Voice Feedback & Token Rewards:** Supports emotional regulation and motivation  
- **Catch the Comet (Focus Training):** Builds sustained attention with measurable metrics  
- **Low-Sensory UI:** Reduces overwhelm and improves task completion  
- **Clinical Logs:** Tracks time, errors, distractions, attempts, focus %, reaction time  
- **Future:** Therapist dashboard to toggle rooms, adjust sensory load, export logs, and set tasks
